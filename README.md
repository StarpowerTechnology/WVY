***Perpetual Autonomous Simulation***

*Also reffered to as "Agent Universes" but for fully independent agents.*

***What is WVY?***

*This system is designed for multiple agents to have their own self driven goals & complete tasks beyond the users request for signal-based decision making within a simulation runtume rather than the traditional reactive/proactive "helpful assistant". The goal with Starpower Autonomy is to create ateam of cognitive architectures that have adaptable memory, socialize, create, review eachothers work & make every decision on their own.*

*As of April 24th, 2026, this repo is just the foundation for a multi-perspective groupchat. Unlike other multi agent systems, this is simply deisgned for agents to speak to each other instead of all agents acting as one mind. The agents reply, set their own sleep timer & wake up to the chat to respond again*
